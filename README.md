# Power of 2

## Gist
Given an integer n, return true if it is a power of two. Otherwise, return false.

An integer n is a power of two, if there exists an integer x such that n == 2x.


## Examples
Example 1:

Input: n = 1
<br/>
Output: true
<br/>
Explanation: 2^0 = 1
<br/>

Example 2:

Input: n = 16
<br>
Output: true
<br/>
Explanation: 2^4 = 16
<br/>

Example 3:

Input: n = 3
<br/>
Output: false
